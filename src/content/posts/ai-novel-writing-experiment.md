前面提到过，我有个想法：看小说不如自己写小说。

---

## 为什么想自己写

因为有时候觉得这地方应该这么写，而不是像作者那么写

因此萌生了这个想法。

所以我想，既然有想法，为什么不写？反正现在有 AI，我又是 AI 应用工程师，这不是天时地利人和吗？于是我决定：搭一个写小说的 Agent。

---

## Agent 的搭建思路

说实话，这个 Agent 搭起来没费多少时间，毕竟是本职工作。

整体结构大概是这样：

- **大纲模块**：给定世界观、主角设定、核心爽点，生成章节大纲
- **写作模块**：根据大纲逐章生成内容，带上前文摘要做上下文
- **记忆模块**：维护一个人物表、境界表、道具表，防止前后矛盾
- **润色模块**：生成完之后再过一遍，优化语言节奏

看上去很完美对吧。

我自己都感动了。

---

## 然后问题来了

**第一个问题：单章质量确实不错。**

真的，我没吹。单独看一章，文笔流畅，爽点到位，打斗描写有层次，情绪渲染也在线。我拿给朋友看，他问我这是从哪本书摘的。

我说是 AI 写的。

他沉默了三秒，说：比某些签约作者强。

我飘了大概五分钟。

---

**然后飘完，我开始连章看。**

然后我的表情就变了。

**问题一：剧情断章。**

主角上一章刚刚打了一架，受了重伤，生死未卜，下一章直接在城里逛街买东西，脸上没有半点伤，神情悠哉。

我去检查上下文，发现摘要模块把"受重伤"这个信息给丢了。

行，我加强摘要。

加强完，下一次生成，主角受伤被记住了，但是他击败的那个反派……复活了，以全新的面目出现在第二十章，完全忘记自己已经死过一次。

我：？

**问题二：境界混乱，简称"境界薛定谔"。**

我设定了一套境界体系：练气、筑基、金丹、元婴、化神……挺经典的修仙体系。

前十章主角是练气期，突破筑基花了三章写得很热血。

到了第十五章，一个路人甲出场，介绍词写的是"金丹期强者"，下一句话他被主角一拳打飞了。

主角当时是筑基期初阶。

我盯着屏幕看了很久，决定这个锅不能让主角背，一定是路人甲体质有问题。

但 Agent 不这么认为，它继续在后面的章节里随机给各种角色安排境界，忽高忽低，完全不参考我设定的那张境界表。

我检查了一下，发现它根本没有认真读那张表，只是在生成的时候凭感觉填了一个听上去厉害的词。

专业术语叫：幻觉。

---

## 我尝试的那些解法

我不是没努力，我试了很多方法：

**方法一：加长 大纲设定，把重要设定都塞进去。**

结果：Prompt 太长，模型开始选择性失忆，越到后面忘得越多，像极了我背单词的状态。

**方法二：RAG 检索设定库，每次写作前先召回相关设定。**

结果：召回内容跟生成内容经常对不上，召回了"金丹期强者有哪些特征"，生成的时候依然把金丹期写成了炮灰。

**方法三：写完一章强制做一次"事实核查"，让另一个模型来审查前后矛盾。**

结果：核查模块发现了矛盾，主写模块在下一章继续犯同样的错误，两个模型形成了一种奇妙的对话——一个发现问题，一个无视问题，相安无事，各司其职。

**方法四：降低自由度，给每一章都写详细的分章大纲，让模型只负责扩写。**

结果：这个方法效果最好，章节之间的连贯性大幅提升。

但问题是——我需要把每一章都想清楚才能写详细大纲，那我自己直接写不就行了？

---

## 最终结论

折腾了一段时间之后，我得出了一个朴素的结论：

**Agent 写小说，只能是副驾驶，不能是司机。**

你得有整体架构，有清晰的世界观，有每一卷的核心冲突，有每一章的走向——然后拿这些去喂 AI，让它帮你把骨架填成肉。

这样的话，速度会快很多，质量也能保证。

但如果你想从零开始，把一个模糊的"我想写个玄幻"的念头交给 Agent，让它全流程自动跑出一本书……

你会得到一本境界薛定谔、反派不死、主角时不时失忆的超展开作品。

也许也有人喜欢看，谁知道呢。

## 但是，我发现了新方向

就在我准备暂时搁置这件事的时候，我发现了一个新思路。

把 Claude Code 接入 DeepSeek，然后做一个专门用于写小说的 Claude Code Skill。

直接在 Claude Code 里面跑。

理论上，Claude Code 本身的上下文管理、任务拆解、文件读写能力比我自己搭的那个 Agent 强多了，而且操作起来更灵活，可以直接把世界观、人物设定、已写章节都作为文件丢进去，让它读完再写。

加上 DeepSeek 的中文能力，写出来的文字质量可能会更贴合玄幻风格。

我还没试，纯理论阶段。

但这个想法让我感觉很有搞头——就像每次开始一个新项目之前的那种兴奋感，干劲十足，充满幻想，完全没有考虑后面会踩什么坑。

这种感觉很好。

先这样。等我试完了再来汇报。

目前还不是很想去尝试，我应该又变懒惰了，可恶呀

三千七百四十二年。

七月初三。

青云镇云家，以星辰功法立族。子弟年满十六，需入祖祠凝聚星核。成则一步登天，败则沦为底层。前世他失败后，被发配矿场做了三年苦役，受尽欺凌。直到某次矿洞坍塌，他侥幸未死，反而在废墟深处寻到半部残缺功法，这才重新踏上修行路。

果然。

那道暗伤，从这一刻就已经埋下了。

前世他懵懂无知，直到百年后渡劫身死才察觉。这一世……他绝不会再犯同样的错误。

云墨慢慢坐起身。四肢酸软无力，丹田空空如也——那是凝聚星核失败后的典型症状。寻常人遭此打击，多半一蹶不振。

这一世……

他对着晨光，对着这个崭新而残酷的世界，一字一顿地吐出誓言：

“我要换个活法。”
